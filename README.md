# Birthday Gift Website

This project is a static GitHub Pages-ready birthday website.

## Experience

- The site starts as a sealed envelope.
- Clicking the envelope opens a birthday letter with the message `Happy Birthday liebs Schnuegli`.
- A portrait card appears beside the letter.
- The gift then dodges her until the 28th attempt, quietly tying the challenge to her 28th birthday.
- The final reveal is a voucher for `5 x Pilates` at the studio of her choice, such as BLG Sports in Zurich.

## Add her photo

Place her image in the project folder and name it:

`girlfriend-photo.jpg`

The site already looks for that file automatically.

Note:
I styled the image like a soft sticker/cutout card inside the page. If you want a true transparent-background cutout, remove the background from the photo first and then use that exported image here.

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, and optionally `girlfriend-photo.jpg`.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select your main branch and the `/ (root)` folder.
6. Save and wait for the GitHub Pages link to appear.

## Personal touches you can still change fast

- Edit the letter text in `index.html`.
- Edit the final voucher copy in `index.html`.
- Change the number of tries in `script.js` if you ever want a shorter or longer version.
